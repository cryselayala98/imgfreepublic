Este es un proyecto tipo red social como Imgur

herramientas: NodeJS, MongoDB, uso de promesas y peticiones Ajax

1. que las imgs que se suban no pasen de cierto peso(no subir imagenes tan pesadas)
2. una autenticación(para que no todos los usuarios puedan eliminar cualquier imagen)
3. notificaciones - con web Sockets
4. un chat en tiempo real
5. usar Ajax para que al comentar no haya la necesidad de refrescar toda la página
6. validar el texto que escriba el usuario en los comentarios
